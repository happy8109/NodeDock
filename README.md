# NodeDock

**NodeDock** 是一款基于 .NET Framework 4.6.2 开发的轻量级 Node.js 应用管理工具。它旨在解决 Windows 环境下开发/部署多个 Node.js 程序时出现的 CMD 窗口堆积、管理混乱以及版本冲突等问题。

## 🚀 主要特性

- **后台驻留**：将 Node.js 应用隐藏在后台运行，消除桌面黑窗口，支持最小化至任务栏托盘。
- **多版本集成**：内置多版本 Node.js 运行环境切换，不受系统全局 Node 版本的限制。
- **分散目录支持**：支持管理存储在操作系统任何位置的 Node.js 项目。
- **状态监控**：直观展示每个应用的运行状态（运行中、已停止、异常退出）。
- **实时预览**：支持在不干扰程序运行的情况下，随时查看程序的终端控制台实时输出。
- **一键托管**：支持开机自启（配置中）以及所有程序的一键启动与停止。

## 📂 目录结构

```text
NodeDock/
├── runtimes/     # 放置不同版本的免安装 Node.js 执行程序
├── config/      # 存放配置文件
├── docs/        # 开发文档与技术指南
└── NodeDock.exe # 主运行程序
```

## 📖 快速开始

### 1. 准备运行环境
在 `runtimes` 文件夹下按版本号建立子文件夹（如 `v18.16.0`），并将下载好的 Node.js 免安装包内容（包含 `node.exe`、`npm.cmd` 及 `node_modules` 文件夹）放入其中。

### 2. 添加应用
运行 NodeDock，点击“添加”按钮，选择您的 Node 项目路径、入口脚本并关联对应的 Node 版本。

### 3. 运行管理
通过主界面或托盘右键菜单，轻松启动或停止您的项目。
