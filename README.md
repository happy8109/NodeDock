# NodeDock

**NodeDock** 是一款基于 .NET Framework 4.6.2 开发的轻量级 Node.js 应用管理工具。它旨在解决 Windows 环境下开发/部署多个 Node.js 程序时出现的 CMD 窗口堆积、管理混乱以及版本冲突等问题。

## 🚀 主要特性

### 应用管理
- **后台驻留**：将 Node.js 应用隐藏在后台运行，消除桌面黑窗口，支持最小化至任务栏托盘
- **分散目录支持**：支持管理存储在操作系统任何位置的 Node.js 项目
- **状态监控与资源看板**：直观展示应用运行状态，**实时监控每个进程的 CPU 和内存占用情况**
- **进程守护**：支持应用异常退出后自动重启，确保服务持续可用
- **扁平化 UI**：采用现代化的自定义标签系统，告别传统 Tab 控件的厚重感
- **实时预览**：在不干扰程序运行的情况下，随时查看程序的终端控制台实时输出
- **一键托管**：支持所有程序的一键启动与停止

### 运行时版本管理
- **多版本集成**：内置多版本 Node.js 运行环境切换，不受系统全局 Node 版本限制
- **智能版本推荐**：自动探测项目 `package.json` 中的 `engines.node` 字段，智能推荐符合项目需求的版本
- **版本下载**：支持在线下载 Node.js LTS 版本，可切换官方源或淘宝镜像加速
- **默认版本设置**：可设置默认运行时版本，新建应用时自动关联
- **版本删除**：支持删除不再需要的版本，删除前自动检查是否有应用正在使用

### 系统设置
- **开机自启动**：支持随 Windows 启动自动运行
- **镜像源切换**：支持官方源 (nodejs.org) 和淘宝镜像 (npmmirror.com)

## 📂 目录结构

```text
NodeDock/
├── runtimes/          # 存放不同版本的 Node.js 运行时
│   ├── v22.11.0/
│   ├── v24.12.0/
│   └── ...
├── config/
│   └── settings.json  # 配置文件（应用列表、偏好设置）
├── docs/
│   └── DEVELOPMENT_GUIDE.md  # 开发文档
├── Properties/
│   └── AssemblyInfo.cs       # 版本信息
└── NodeDock.exe       # 主程序
```

## 📖 快速开始

### 1. 下载运行时
运行 NodeDock，点击 **设置** → **下载新版本**，选择需要的 Node.js 版本进行下载安装。

### 2. 添加应用
点击 **添加** 按钮，选择您的 Node 项目路径、入口脚本并关联对应的 Node 版本。

### 3. 运行管理
通过主界面或托盘右键菜单，轻松启动或停止您的项目。
